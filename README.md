# vue-tailwind-template

A template for making web app with Vue.js and Tailwind.

## How to use

### Install all dependencies

```
$ yarn install
```

### Compiles and hot-reloads for development
```
$ yarn dev
```

### Compiles and minifies for production
```
$ yarn build
```

### Lints and fixes files
```
yarn run lint
```
